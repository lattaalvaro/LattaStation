# 🎮 LattaStation

**Emulador de PlayStation 1 para Android — rápido, limpio y con estilo retro.**

*Revive tus clásicos de PS1 con una interfaz moderna pensada para móvil.*

</div>

---

## ✨ Características

- 🏠 **Pantalla de inicio inteligente** — acceso rápido a tus juegos más recientes y más jugados
- 📚 **Biblioteca personal** — organiza tus ROMs por nombre, fecha, tiempo jugado o tamaño
- 🏆 **Sistema de logros** — desbloquea metas como *Primer arranque*, *Maratonista*, *Leyenda PS1* y más
- 🎨 **Múltiples temas** — Oscuro, Claro, Material You, Retro PlayStation y AMOLED
- 🖥️ **Modos de pantalla** — 4:3, Estirar, Entero, Ancho y Alto
- 🔬 **Resolución interna GPU escalable** — desde 1× (320×240) hasta 6× (1920×1440)
- ☁️ **Sincronización en la nube** — copia de seguridad de saves, memory cards y ajustes
- ⭐ **Favoritos** — marca los juegos que más quieres tener a mano

## 📥 Instalación

### Requisitos
- Android 8.0 (Oreo) o superior
- Arquitectura ARM64 / ARM32
- Almacenamiento suficiente para tus ROMs de PS1

### Pasos

1. Ve a la sección [**Releases**](../../releases) de este repositorio
2. Descarga el archivo `.apk` de la versión más reciente
3. En tu Android, activa **"Instalar apps de fuentes desconocidas"**
   - *Ajustes → Seguridad → Fuentes desconocidas*
4. Abre el `.apk` descargado e instala
5. ¡Listo! Abre LattaStation y escanea tu carpeta de ROMs

> ⚠️ **Aviso legal:** LattaStation es un emulador. No distribuye ni incluye ROMs, BIOS ni ningún contenido con derechos de autor. El usuario es responsable de obtener las ROMs de juegos que posee legalmente.

---

## 🚀 Primeros pasos

```
1. Abre la app → ve a Ajustes (⚙️)
2. Toca "Escanear carpeta de ROMs"
3. Selecciona la carpeta donde tienes tus archivos .bin / .cue / .iso
4. La biblioteca se actualiza automáticamente
5. ¡Pulsa un juego y empieza a jugar!
```
